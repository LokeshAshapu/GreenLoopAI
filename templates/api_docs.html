REST API v1.0

API Reference Documentation

Integrate environmental reports, maps, and AI analysis outputs directly with external municipal dashboards and civilian scripts.

Core Endpoints 1. Authentication (JWT) 2. Environmental Reports 3. Map Points (GeoJSON) 4. Community Verification

1. Authentication (SimpleJWT)

To access secure API routes, requests must include a Bearer access token in the authorization header.

POST /api/token/

Exchange username and password credentials for an access and refresh token.

# Payload request {"username": "Lokesh", "password": "SecurePassword123!"}
# Response (Status 200 OK) {"access": "eyJhbGciOi...", "refresh": "eyJhbGciOi..."}

2. Environmental Reports

Upload pictures and details to run NVIDIA VLM hazard scans.

GET /api/reports/

Retrieve list of all uploaded reports and nested AI diagnostics.

POST /api/reports/

Create a report. Must submit as **multipart/form-data** containing: `title`, `description`, `category`, `latitude`, `longitude`, and `image` (binary file).

# Response (Status 201 Created) { "id": 1, "title": "Severe Plastic Waste Accumulation", "category": "plastic_waste", "latitude": "40.71280000", "longitude": "-74.00600000", "status": "submitted", "image": "/media/reports/plastic_waste.png", "ai_analysis": { "confidence_score": 98.40, "severity_score": 8.20, "environmental_risk_index": 7.90, "recommended_action": "Organize volunteer cleanup...", "health_risk_summary": "Risk of microplastics..." } }

3. Map Points (GeoJSON)

Public unauthenticated endpoint to load map pins.

GET /api/reports/geojson/
# Response (Status 200 OK) { "type": "FeatureCollection", "features": [ { "type": "Feature", "geometry": { "type": "Point", "coordinates": [-74.006, 40.7128] }, "properties": { "id": 1, "title": "Severe Plastic Waste Accumulation", "status": "verified", "severity": 8.2, "image_url": "/media/reports/plastic_waste.png" } } ] }

4. Community Verification

Register votes on submitted reports.

POST /api/reports/<report_id>/verify/
# Payload request {"is_valid": true, "comments": "Verified. Clean up scheduled."}
